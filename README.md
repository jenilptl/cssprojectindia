# Incredible India

A static web project showcasing the diversity, beauty, and rich heritage of India. This project explores Indian culture, landmarks, cuisine, and more through interactive web pages.

## Features

- **Home**: Overview of India's diversity and highlights.
- **Landmarks**: Information on iconic monuments like the Taj Mahal and Red Fort.
- **Cuisine**: A journey through North and South Indian dishes and desserts.
- **Culture**: Insights into festivals, languages, and traditional attire.
- **Contact Us**: Student registration form and inquiry section.

## Technologies Used

- HTML5
- CSS3
- Audio/Media elements

## Project Structure

- `index.html`: Main landing page
- `landmarks.html`: Landmark information
- `cuisine.html`: Culinary highlights
- `culture.html`: Cultural insights
- `contact.html`: Contact and registration page
- `*.css`: Styling for each respective page
- `mixkit-relaxing-in-nature-522.mp3`: Background audio for a relaxing experience
